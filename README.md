# Tech News APP developed by flutter_web_view
Tech News developed by Flutter (Dart) it is a simple WebView that run on Android & iOS.

It reads the news from these websites to show:

https://gadgetnews.net/

https://digiato.com/

https://www.zoomit.ir/

https://click.ir
